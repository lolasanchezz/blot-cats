# stipple cats
## a homage to one of my first programming experiences

one of the first real coding projects i ever took on was to redeem the blot kit
from hack club, which required coding some piece of art in the online blot gui to 
have the blot make. i spent around 15 hours on it, and it was the first time i 
enjoyed programming as a creative excersize to make something new just for the sake of making it. 
now, with a full time job as a software engineer, i found the original 
code and put it all on one html file, stealing some code from claude and the original
blot libraries to make it work through html canvas api. all it is is a pseudo-random
stipple cat generator, with varying eyes, ears, and dot densities.
